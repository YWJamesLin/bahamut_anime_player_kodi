# bahamut_anime_player_kodi
Bahamut anime player addon of kodi

# Kodi 巴哈姆特動畫瘋播放器插件
在Kodi上看動畫瘋，此附加元件限購買動畫瘋去廣告功能者使用，請在安裝以後先在設定中指定帳號密碼

## Requirements
python2 plugins :
  requests
  bs4
  pickle
  urlparse

## Usage :
Just
```bash
zip -r plugin_video_anigamer.zip ./plugin.video.anigamer
```
and install this addon package.
